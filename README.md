# OpenConnect

### Server installation command:

#### Before running the code, apply this command:

- sudo su

- cd

- wget https://raw.githubusercontent.com/SmartLogics-c/OpenConnect/master/ocserv.sh && chmod +x /root/ocserv.sh && bash /root/ocserv.sh

</br>
